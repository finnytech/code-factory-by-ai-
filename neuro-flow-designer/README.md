# NeuroFlow Designer ✦

An advanced, visually stunning, web-based IDE, workflow node editor, and 2D Canvas simulator for AI agent swarm dynamics.

## Features

- **Visual Node Flowchart Editor**: Construct multi-agent collaborative workflows using drag-and-drop mechanics. Link agents (Planner, Coder, Critic) and integration tools (Search, Compile, Publisher) together.
- **2D Swarm Arena Physics Simulation**: Watch agents act as particles steering towards tasks, colliding to process tasks, and bursting on completion.
- **Dynamic Performance Diagnostics**: Live telemetry logs, calculated execution queue size, avg latencies, and token cost accumulation ($/TPS) based on simulated Gemini model fees.
- **Responsive Cyberpunk GUI**: Sleek glassmorphism theme with neon glowing accents, optimized grid sizing, and keyframe-based node status updates.

## Getting Started

Simply open `index.html` in any modern web browser to run the workspace:

```bash
# Serve locally
npm install -g local-web-server
ws -d ./neuro-flow-designer
```

Or run directly by double-clicking the `index.html` file on your filesystem.
